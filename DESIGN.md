# No Reaction — 游戏设计文档

## 概述

**No Reaction** 是一款以高中化学为主题的双人联机卡牌对战游戏。玩家轮流向公共反应区打出离子牌，目标是不让反应区发生化学反应。迫使反应发生的玩家判负。

---

## 核心规则

### 游戏目标

保持反应区"无反应"（No Reaction）状态。当某玩家打出的离子牌与反应区内已有离子发生反应（沉淀或气体），且无法用道具牌解除时，该玩家判负。

**特殊情况——中和反应**：H⁺ 与 OH⁻ 相遇时不判定失败，而是直接将这两张牌同时移入弃牌区（自动中和，无需道具）。

### 回合流程

1. **摸牌**：从牌堆摸 **2 张**牌
   - 例外——若玩家**既无离子牌也无催化剂**，触发**惩罚阶段**（见下）
2. **出牌阶段**：打出 1 张离子牌，**或**打出催化剂（催化剂替代离子牌，不入反应区）
   - 受催化剂影响时（本回合 `forcedIons=2`）：必须连续打出 **2 张离子牌**（手牌不足则尽力而为）；此时不可用催化剂代替
3. **道具阶段**：可连续打出任意数量的防守/中性道具（过滤、加热、萃取、中和、搅拌）
4. **反应判定**：玩家确认结束道具阶段 → 扫描反应区，有沉淀或气体反应 → **直接判负**
5. 无反应 → 回合结束，换对手

### 惩罚阶段

若玩家**回合开始时**手牌中**既无离子牌也无催化剂**：

- 额外摸 **2 张**牌（替代正常摸牌）
- 手牌中所有道具牌转给对手
- 直接进入道具阶段（跳过出牌阶段）

### 初始设置

- 每位玩家初始手牌：**7 张**
- 游戏人数：**2 人**（后续扩展多人）

## 卡牌系统

### 离子牌

**阳离子（9 种）**

| 离子 | 符号 | 张数 |
|------|------|------|
| 钡离子 | Ba²⁺ | 2 |
| 钙离子 | Ca²⁺ | 2 |
| 铜离子 | Cu²⁺ | 2 |
| 铁离子 | Fe³⁺ | 2 |
| 银离子 | Ag⁺ | 2 |
| 铵根离子 | NH₄⁺ | 2 |
| 氢离子 | H⁺ | 4 |
| 钠离子 | Na⁺ | 2 |
| 铝离子 | Al³⁺ | 2 |

**阴离子（8 种）**

| 离子 | 符号 | 张数 |
|------|------|------|
| 硫酸根 | SO₄²⁻ | 2 |
| 碳酸根 | CO₃²⁻ | 2 |
| 氢氧根 | OH⁻ | 4 |
| 氯离子 | Cl⁻ | 2 |
| 硝酸根 | NO₃⁻ | 2 |
| 磷酸根 | PO₄³⁻ | 2 |
| 硫离子 | S²⁻ | 2 |
| 碳酸氢根 | HCO₃⁻ | 2 |

> 所有离子数量在 `config/cards.js` 中集中管理，方便测试时调整。

---

### 道具牌

| 道具 | 张数 | 类型 | 效果 |
|------|------|------|------|
| 过滤 | 3 | 防守 | 移走反应区中一对沉淀离子（一个阳离子 + 一个阴离子）；一个阳离子对应多个阴离子时，每次只能移走一对 |
| 加热 | 3 | 防守 | 移走反应区中一个参与气体反应的离子（仅限气体反应触发时使用） |
| 萃取 | 2 | 防守 | 将反应区中指定的一个离子取回自己手牌，可用于任意类型反应 |
| 中和 | 2 | 防守 | 手动移走反应区中一个 H⁺ 或一个 OH⁻（仅限 H⁺/OH⁻ 参与的反应，不能处理 NH₄⁺ + OH⁻） |
| 搅拌 | 2 | 中性 | 清空整个反应区，所有离子移入弃牌堆 |
| 催化剂 | 2 | 攻击 | 替代离子牌打出（不向反应区加入离子）。对手**下回合**须连续打出 **2 张离子牌**（手牌不足则尽力而为）；被强制出牌时不可再用催化剂代替。 |

> 道具牌数量同样在 `config/cards.js` 中管理。

## 反应判定表

打出离子后，与反应区内所有离子逐一配对检测。

### 中和反应（自动处理，不判负）

| 组合 | 处理方式 |
|------|----------|
| H⁺ + OH⁻ | 两张牌同时自动移入弃牌区 |

### 沉淀反应（触发判定）

| 阳离子 | 阴离子 | 产物 | 备注 |
|--------|--------|------|------|
| Ba²⁺ | SO₄²⁻ | BaSO₄↓ | 白色沉淀 |
| Ba²⁺ | CO₃²⁻ | BaCO₃↓ | 白色沉淀 |
| Ca²⁺ | SO₄²⁻ | CaSO₄↓ | 白色沉淀 |
| Ca²⁺ | CO₃²⁻ | CaCO₃↓ | 白色沉淀 |
| Cu²⁺ | OH⁻ | Cu(OH)₂↓ | 蓝色沉淀 |
| Fe³⁺ | OH⁻ | Fe(OH)₃↓ | 红褐色沉淀 |
| Al³⁺ | OH⁻ | Al(OH)₃↓ | 白色沉淀 |
| Al³⁺ | CO₃²⁻ | — | 双水解，不共存，触发判定 |
| Ag⁺ | Cl⁻ | AgCl↓ | 白色沉淀 |
| Ag⁺ | SO₄²⁻ | Ag₂SO₄↓ | 微溶，白色沉淀 |
| Ba²⁺ | PO₄³⁻ | Ba₃(PO₄)₂↓ | 白色沉淀 |
| Ca²⁺ | PO₄³⁻ | Ca₃(PO₄)₂↓ | 白色沉淀 |
| Ag⁺ | PO₄³⁻ | Ag₃PO₄↓ | 黄色沉淀 |
| Cu²⁺ | S²⁻ | CuS↓ | 黑色沉淀 |
| Ag⁺ | S²⁻ | Ag₂S↓ | 黑色沉淀 |

> 注：Fe³⁺ + S²⁻ 实为氧化还原反应（不生成沉淀），不纳入判定表。

### 气体反应（触发判定）

| 组合 | 产物 |
|------|------|
| H⁺ + CO₃²⁻ | CO₂↑ |
| H⁺ + HCO₃⁻ | CO₂↑ |
| H⁺ + S²⁻ | H₂S↑ |
| NH₄⁺ + OH⁻ | NH₃↑ |

> 反应判定表存放在 `js/reactions.js` 中，可独立更新。

## 技术方案

| 模块 | 方案 | 说明 |
|------|------|------|
| 前端 | 纯 HTML/CSS/JS | 无需构建工具，浏览器直开 |
| 联机 | LeanCloud 实时通信 | 国内服务，不被墙，免费套餐 |
| 托管 | Cloudflare Pages | 国内可访问，免费，无需购买域名 |
| 房间机制 | 6 位房间码 | 创建/加入房间，共享游戏状态 |
